# SBI Financial Co-Pilot - Local Setup Guide\n\n## Prerequisites\n\nBefore you start, ensure you have:\n- **Node.js** (v16+): [Download](https://nodejs.org/)\n- **Python** (v3.9+): [Download](https://www.python.org/)\n- **MongoDB** (Local or Atlas): [Download](https://www.mongodb.com/try/download/community) or [Atlas](https://www.mongodb.com/cloud/atlas)\n- **Git**: [Download](https://git-scm.com/)\n\n---\n\n## Step 1: Clone Repository\n\n```bash\ngit clone https://github.com/kumar-ayush30/sbi-financial-co-pilot.git\ncd sbi-financial-co-pilot\n```\n\n---\n\n## Step 2: Setup Backend (Python/FastAPI)\n\n### 2.1 Create Python Virtual Environment\n\n```bash\ncd backend\npython -m venv venv\n\n# Activate virtual environment\n# On Windows:\nvenv\\Scripts\\activate\n# On macOS/Linux:\nsource venv/bin/activate\n```\n\n### 2.2 Install Dependencies\n\n```bash\npip install -r requirements.txt\n```\n\n### 2.3 Create `.env` File\n\nCreate `backend/.env` with:\n\n```env\n# MongoDB Configuration\nMONGO_URL=mongodb://localhost:27017\nDB_NAME=sbi_copilot\n\n# JWT Configuration\nJWT_SECRET=your-super-secret-key-change-in-production-12345\nACCESS_TOKEN_EXPIRE_MINUTES=15\nREFRESH_TOKEN_EXPIRE_DAYS=7\n\n# CORS Configuration\nCORS_ORIGINS=http://localhost:3000,http://localhost:3001\n\n# Optional: Emergent LLM (for AI features)\nEMERGENT_LLM_KEY=your-emergent-key-if-available\n```\n\n### 2.4 Start MongoDB\n\n#### Option A: Local MongoDB\n\n```bash\n# Windows\nmongod\n\n# macOS (if installed via Homebrew)\nbrew services start mongodb-community\n\n# Linux\nsudo systemctl start mongod\n```\n\n#### Option B: MongoDB Atlas (Cloud)\n\n1. Go to [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)\n2. Create account → Create cluster → Get connection string\n3. Update `MONGO_URL` in `.env`:\n\n```env\nMONGO_URL=mongodb+srv://username:password@cluster.mongodb.net/?retryWrites=true&w=majority\n```\n\n### 2.5 Start Backend Server\n\n```bash\ncd backend\npython -m uvicorn server:app --reload --host 0.0.0.0 --port 8000\n```\n\n**Backend running at:** `http://localhost:8000`\n\n---\n\n## Step 3: Setup Frontend (React)\n\n### 3.1 Install Dependencies\n\n```bash\ncd frontend\nnpm install\n```\n\n### 3.2 Create `.env` File\n\nCreate `frontend/.env` with:\n\n```env\nREACT_APP_API_URL=http://localhost:8000/api\n```\n\n### 3.3 Start Frontend Development Server\n\n```bash\nnpm start\n```\n\n**Frontend running at:** `http://localhost:3000`\n\n---\n\n## Step 4: Test the Application\n\n### 4.1 Access the App\n\nOpen browser and go to: **http://localhost:3000**\n\n### 4.2 Register New Account\n\n**Step 1: Fill Details**\n- Full name: `Test User`\n- Email: `test@example.com`\n- Phone: `9999999999`\n- Password: `Password123` (must have 8+ chars, uppercase, digit)\n- Age: `28`\n- Monthly Income: `65000`\n- Risk Tolerance: `Medium`\n- Financial Goal: `Build emergency fund`\n\n**Step 2: Verify Phone with OTP**\n- Click \"Send OTP\"\n- **Demo OTP: `000000`** (or use `123456`)\n- Enter and click \"Verify & Continue\"\n\n**Step 3: Complete KYC**\n- PAN: `ABCDE1234F` (10 chars)\n- Aadhaar Last 4: `1234`\n- SBI Account Last 4: `5678` (optional)\n- Click \"Create account & see my dashboard\"\n\n### 4.3 Load Demo Data\n\nAfter registration:\n1. Click \"Use Demo Dataset\"\n2. ~75 realistic Indian transactions loaded ✅\n\n### 4.4 Test Features\n\n- **Dashboard**: View balance, income, expenses, health score\n- **Transactions**: See all transactions with fraud detection scores\n- **Financial Health**: View detailed health metrics\n- **Recommendations**: Get savings and product recommendations\n- **Wealth Simulator**: Project future wealth (SIP calculator)\n- **AI Chat**: Ask financial questions\n- **Settings**: Upload CSV, reset data, delete account\n\n---\n\n## Step 5: Test API Endpoints\n\n### Using Postman or cURL\n\n#### 1. Register\n```bash\ncurl -X POST http://localhost:8000/api/auth/register \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"full_name\": \"Test User\",\n    \"email\": \"test@example.com\",\n    \"phone\": \"9999999999\",\n    \"password\": \"Password123\",\n    \"age\": 28,\n    \"monthly_income\": 65000\n  }'\n```\n\n#### 2. Send OTP\n```bash\ncurl -X POST http://localhost:8000/api/auth/otp/send \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"phone\": \"9999999999\"}'\n```\n\nResponse:\n```json\n{\n  \"sent\": true,\n  \"demo_otp\": \"000000\",\n  \"message\": \"OTP sent (demo mode). Use 000000 or 123456 for testing.\"\n}\n```\n\n#### 3. Verify OTP\n```bash\ncurl -X POST http://localhost:8000/api/auth/otp/verify \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"phone\": \"9999999999\", \"otp\": \"000000\"}'\n```\n\n#### 4. Check Fraud Detection\n```bash\ncurl -X POST http://localhost:8000/api/fraud/check-transaction \\\n  -H \"Content-Type: application/json\" \\\n  -H \"Authorization: Bearer YOUR_ACCESS_TOKEN\" \\\n  -d '{\n    \"transaction_date\": \"2026-06-29\",\n    \"amount\": 500000,\n    \"merchant_name\": \"Bitcoin Exchange\",\n    \"category\": \"Cryptocurrency\",\n    \"transaction_type\": \"debit\"\n  }'\n```\n\nResponse:\n```json\n{\n  \"fraud_score\": 85,\n  \"risk_level\": \"HIGH\",\n  \"flags\": [\"risky_category: Cryptocurrency\"],\n  \"reason\": \"Multiple fraud indicators detected...\"\n}\n```\n\n---\n\n## Troubleshooting\n\n### Backend Issues\n\n#### MongoDB Connection Error\n```\nError: connect ECONNREFUSED 127.0.0.1:27017\n```\n\n**Fix:**\n- Ensure MongoDB is running: `mongod` (Windows) or `brew services start mongodb-community` (macOS)\n- Check `MONGO_URL` in `.env`\n\n#### Import Error: emergentintegrations\n```\nModuleNotFoundError: No module named 'emergentintegrations'\n```\n\n**Fix:** This is OK - the app falls back to heuristic responses. Optional for full AI features.\n\n#### Port 8000 Already in Use\n```bash\n# Use different port\npython -m uvicorn server:app --reload --port 8001\n```\n\n### Frontend Issues\n\n#### Port 3000 Already in Use\n```bash\n# Use different port\nPORT=3001 npm start\n```\n\n#### API Connection Error\n```\nFailed to fetch http://localhost:8000/api/...\n```\n\n**Fix:**\n- Ensure backend is running on port 8000\n- Check `REACT_APP_API_URL` in `.env`\n- Verify CORS settings in `backend/.env`\n\n#### Blank Login Page\n```bash\n# Clear cache and reinstall\nrm -rf node_modules package-lock.json\nnpm install\nnpm start\n```\n\n---\n\n## Project Structure\n\n```\nsbi-financial-co-pilot/\n├── backend/\n│   ├── server.py              #
